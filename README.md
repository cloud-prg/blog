## 介绍
基于next.js、ts、tailwindcss，配合个人开发的nestjs后端，实现了一个简单的博客系统。小而全，包括 `docker`、`jenkins`来跑通自动部署全流程。

## 开始
```bash
pnpm install
pnpm run dev
```

## 构建
```bash
pnpm run build
```
